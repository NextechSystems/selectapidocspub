# PaymentReconciliation

## PaymentReconciliation

### Overview
A list of payments, adjustments, and refunds. Can be insurance or self-pay.

### *Search*
Searches for all  based on the given search criteria.

#### HTTP Request 
`GET /PaymentReconciliation?{parameters}`

#### Parameters
| Name | Located in | Description | Required | Initial Version |
| ---- | ---------- | ----------- | -------- | --------------- |
| identifier | query or uri | The unique value assigned to each PaymentReconciliation |  No | _14.1_ |
| location | query | The location for the PaymentReconciliation. | No | _14.1_ |
| patient | query | The patient the PaymentReconciliation is tied to. | No | _14.1_ |
| created | query | The date the PaymentReconciliation was created. | No | _14.1_ |
| status | query | The status of the PaymentReconciliation. This must be one of the following active, cancelled ( voided ), entered-in-error ( deleted )  | No | _14.1_ |
| payment-date | query |The date of service for the PaymentReconciliation | No | _14.1_ |
| request-provider| query | The provider associated with the PaymentReconciliation. This is a practitioner reference. | No | _14.1_ |
